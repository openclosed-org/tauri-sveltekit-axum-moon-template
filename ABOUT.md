# About | 项目介绍

## One-line Description | 一句话介绍

Desktop-first open source template: Tauri v2 + SvelteKit 5 + Rust workspace + moon + Bun.
面向桌面优先应用的开源模板：Tauri v2 + SvelteKit 5 + Rust workspace + moon + Bun。

## Short Description (GitHub About) | GitHub About 短描述

Business-agnostic desktop app template with Tauri, SvelteKit, and a layered Rust monorepo.
业务无关的桌面应用模板，使用 Tauri + SvelteKit，并提供分层 Rust monorepo 基线。

## Long Description | 长描述

`tauri-sveltekit-axum-moon-template` is a clean, OSS starter for teams that want to build
desktop-first products without being locked into heavy framework opinions.

`tauri-sveltekit-axum-moon-template` 是一个干净的开源起步模板，适合希望构建
桌面优先产品、但又不想被重度框架意见绑住的团队。

It provides:
它提供：

- A Tauri v2 desktop shell.
- A SvelteKit 5 frontend foundation.
- A layered Rust workspace (`domain`, `application`, `shared_contracts`, runtime crates).
- moon + Bun task orchestration and CI quality gates.

- Tauri v2 桌面壳。
- SvelteKit 5 前端基础。
- 分层 Rust workspace（`domain`、`application`、`shared_contracts`、runtime crates）。
- moon + Bun 的任务编排与 CI 质量门禁。

The template intentionally focuses on architecture boundaries and developer workflow,
while leaving business logic, infrastructure choices, and product direction to the adopter.

该模板刻意聚焦在架构边界与开发流程上，把业务逻辑、基础设施选型和产品方向
交给使用方自行定义。

## Intended Audience | 目标人群

- Indie developers shipping desktop apps quickly.
- Small teams wanting clear Rust crate boundaries from day one.
- OSS maintainers who prefer a minimal, composable baseline over batteries-included scaffolds.

- 希望快速交付桌面应用的独立开发者。
- 从第一天就重视 Rust 分层边界的小团队。
- 偏好“轻基线、可组合”而非“全家桶脚手架”的开源维护者。

## Non-Goals | 非目标

- Not a fully opinionated product framework.
- Not a pre-integrated auth/database/payments stack.
- Not a finished backend runtime implementation.

- 不是全套强约束的产品框架。
- 不是预集成认证/数据库/支付的业务脚手架。
- 不是已经完成的后端 runtime 实现。
