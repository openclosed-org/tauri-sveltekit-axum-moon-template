# Docs Archive

This directory stores tracked historical documents that still provide project context but should no longer be treated as current-state truth or default onboarding material.

Use archive when a document is:

1. still worth preserving in Git history as a readable artifact
2. no longer part of the default template-user path
3. no longer the best source of truth for current implementation status

Do not use archive as a scratchpad. Temporary local notes belong in `docs/_local/` instead.
