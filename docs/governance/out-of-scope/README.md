# Out Of Scope Governance Memory

This directory records deliberately rejected default directions so future agents do not relitigate them or reintroduce them as current guidance.

These files are governance memory. They are not runtime evidence and do not prove current behavior.

## When To Add A File

Add a file here when a direction is repeatedly tempting but intentionally out of scope for the default repository path.

A good entry explains:

1. the rejected default direction
2. why it is out of scope
3. what would need to change before reconsidering it
4. related docs or discussions

Do not add temporary deferrals here. Use `docs/_local/**` for active planning notes.
