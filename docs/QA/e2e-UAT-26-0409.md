🚀 启动指南 (!!! justfile 没有对应的启动命令吗? 尽量避免直接使用 moon cargo bun 命令 !!!)

1. 启动后端 API 服务器
   在第一个终端中运行：
   cd /Users/sherlocktang/projects/tauri-sveltekit-axum-moon-template
   cargo run -p runtime_server
   等待看到：
   🚀 Runtime server listening on 0.0.0.0:3001
   Health: http://localhost:3001/healthz
   Ready: http://localhost:3001/readyz
2. 启动前端 Web 应用
   在第二个终端中运行：
   cd /Users/sherlocktang/projects/tauri-sveltekit-axum-moon-template/apps/client/web/app
   bun run dev
   等待看到：
   Local: http://localhost:5173/
3. 启动桌面应用（可选）
   在第三个终端中运行：
   cd /Users/sherlocktang/projects/tauri-sveltekit-axum-moon-template
   just dev-tauri
