# Language Candidates Template

Copy this structure into `docs/_local/language-candidates.md` when a workflow needs scratch vocabulary tracking.

`docs/_local/language-candidates.md` is scratch guidance, not a current rule source. Promote stable terms into `docs/language/**` only after they reduce ambiguity and do not claim runtime behavior.
